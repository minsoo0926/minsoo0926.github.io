KnowGem

Description

  • KnowGem is a focused workspace to connect concepts from documents, visualize relationships, and validate formal logic in one place.

Features

  • Structured Context:
    • Unify scattered sources into one graph context that is easy to navigate in real time.
  • Interactive Graph Visualization:
    • Visualize complex relationships between concepts with interactive graph tools.
  • Formal Validation:
    • Validate reasoning outputs with HOL(Higher Order Logic) expressions to support evidence-based decisions.

Tech Stack

  • Backend: Express.js, Supabase (PostgreSQL)
  • Frontend: React
  • Cloud: AWS
  • DevOps: Terraform, Github Actions

My role in this team

  • Lead the entire engineering progress
  • Full-stack development and system design
  • CI/CD pipeline management
